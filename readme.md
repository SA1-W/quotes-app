<h1>MVC Quote App</h1>

<p>
  Приложение для отображения случайных цитат из базы данных. Логика реализована на чистом PHP с использованием паттерна MVC. 
  Каждое обновление страницы инициирует новый запрос к БД и выводит случайную запись.
</p>

<h3>Технологии:</h3>

<ul>
  <li>PHP (OOP)</li>
   <li>MySQL</li>
   <li>MVC</li>
</ul>

<h3>Примечание:</h3>

<ul>
  <li>Название: БД должна называться quotes-app.</li>
   <li>Импорт: Для работы приложения необходимо импортировать файл .sql из корня проекта в созданную базу.</li>
   <li>Управление: Добавление и редактирование контента производится напрямую через интерфейс phpMyAdmin.</li>
</ul>
